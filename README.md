# Research Hub

A unified research and reading tool that connects web search, personal library, and Zotero into one place.

## What it is
Combines the functions of a research assistant, reading tracker, and writing capture tool. Built for sustained intellectual work across multiple long-term projects.

## Features (planned)
- **Web Research** — search for current topics and recent developments
- **Broad Topic Research** — deep dives across multiple sources
- **Zotero Integration** — cross-reference against existing personal library
- **Reading List** — tracks books list CSV, surfaces next-pick recommendations by category or mood
- **Writing/Speaking Capture Mode** — capture insights tagged by project and theme without breaking research flow

## Connects to
- [`personal-dashboard`](https://github.com/youssefevanw/personal-dashboard) — reading progress and next-pick data
